# 💰 Calculadora de Propinas

Aplicación web desarrollada con **React + TypeScript + Vite** que permite calcular de manera rápida y sencilla el total de consumo en un restaurante, incluyendo la propina.

---

## 📌 Descripción

Este proyecto simula un sistema básico de cálculo de cuentas en un restaurante.
El usuario puede seleccionar productos del menú, ver el subtotal acumulado y aplicar un porcentaje de propina para obtener el total final.

Está orientado a reforzar conceptos clave del desarrollo moderno en React como:

* Manejo de estado con **useReducer**
* Componentes reutilizables
* Tipado fuerte con **TypeScript**
* Separación de lógica mediante helpers
* Arquitectura modular

---

## ⚙️ Funcionalidades

* 🧾 Agregar productos al pedido
* ➕ Incrementar o eliminar items
* 💲 Cálculo automático del subtotal
* 💡 Selección de porcentaje de propina
* 🧮 Cálculo del total final
* 🔄 Reinicio de la orden

## 📁 Estructura del proyecto

* `components` → Componentes UI (formularios, listas, totales)
* `data` → Datos simulados del menú
* `helpers` → Funciones auxiliares
* `reducers` → Lógica del estado global
* `types` → Tipado de TypeScript
